## Notes

- Benchmark on the actual target machine before quoting latency publicly.
- Install `onnxruntime-gpu==1.19.2` on a CUDA host to enable the ONNX Runtime GPU row.
- TensorRT benchmarking should be repeated on the actual Jetson or edge GPU target because engine build options, precision, memory bandwidth, and thermals all affect latency.
